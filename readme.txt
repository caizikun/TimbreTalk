To run this app from a command line on a PC or Mac:
 python tran.py
 
To compile this app for a PC:
 python transetup.py py2exe
The compiled app will be in the directory "dist/" called tran.exe

To pack the app into a single installer file run the Inno Setup script "TranScript.iss".
This will create a file called "installTran.exe"
